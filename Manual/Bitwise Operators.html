On the Subject of Bitwise Operators

Nobody's favorite kind of math. Who even likes math, Anyway?

  • There are 4 total screens on the module
    1. Byte 1
    2. Bitwise operator (AND, OR, XOR, FLIP)
    3. Byte 2
    4. Result input
  • Use the two bytes and the operator to determine the answer.

Should be relatively simple. Do the math below, input the answer.

Here is a table of examples and explanations of each bitwise operator.

Info AND OR XOR FLIP
HOW Going bit by bit, if both bits are 1, the return bit is 1. Otherwise, the return bit is 0. Going bit by bit, if either (or both) bit is 1, the return bit is 1. Otherwise, the return bit is 0. Going bit by bit, if either (but not both) bits are 1, the return bit is 1. Otherwise, the return bit is 0. Basically, ignore the second operator. The answer works regardless. Going bit by bit, the return bit is the opposite.
MATH (bit1 == 1) && (bit2 == 1) (bit1 == 1) || (bit2 == 1) ((bit1 == 1) && (bit2 == 0)) || ((bit1 == 0) && (bit2 == 1)) bit1 = !bit1
